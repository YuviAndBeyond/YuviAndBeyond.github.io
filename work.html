Rapid Identification and Classification of Eccentric BBH Mergers using Machine Learning

Developed a Separable-CNN-based machine learning model to classify eccentric binary black hole (BBH) mergers in real-time. Created a robust data pipeline using Q-transform spectrograms of simulated waveforms injected into non-Gaussian LIGO detector noise. Optimized the model to aid LIGO’s detection pipelines for the upcoming O5 run.

Next

Simulating GW for Symmetric BBH Mergers using PN Approximations

Modeled gravitational wave (GW) signals using quadrupole and post-Newtonian (PN) approximations. Derived PN terms up to 2.5PN, computed inspiral frequencies, and successfully simulated GW150914’s inspiral phase. Used match-filtering techniques to detect the simulated signal, gaining experience in waveform generation and source parameter estimation.

Next

Obtaining Semi-Major Axis for Exoplanets using Machine Learning

Applied regression models (linear regression, decision trees, random forest, SVM, ANN) to predict the semi-major axis of exoplanets using NASA Exoplanet Archive data. Developed a novel classification model and achieved a mean squared error (MSE) of 0.03, demonstrating strong predictive accuracy.

Next

Morphological Classification of Galaxies using Convolutional Neural Network

Developed and optimized a CNN model using the LeNet5 architecture to classify galaxies based on Hubble’s morphological classification. Trained the model on SDSS-16 galaxy images, achieving 89.83% accuracy during training and 70.66% during testing.

Next

Studying the SEIRD Model to Simulate Epidemics using RK4 method

Used the SEIRD epidemiological model to solve coupled differential equations with the Runge-Kutta (RK4) method. Conducted parameter-based analysis and compared SEIRD with other models like SIR and SEIRS to understand disease spread dynamics.

Next

Lattice-Boltzmann Method (LBM) to Understand Fluid Flows

Simulated fluid flow using the D2Q9 lattice model to analyze turbulence effects, such as Kármán vortex streets. Studied the impact of various obstacles on fluid dynamics, demonstrating the potential of LBM for complex fluid systems.
